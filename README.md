# Python-Password_Manager_GUI

In the current times where we use many online web services, I face a lot of issue creating an user account/profile in the due process. Is the account creation process a herculean task? No. But, creating a unique password and keeping a memory of all those one is quiet a herculean task.So, previously I had worked on aPassword generation python script as fun project earlier. 

Now, I used as baseline and was able use Python Tkinter module to create Password Manager GUI which helps me enter the username/email information for either I can create a password manually or with the help of Generate Password interactive button I can automatically generate a unique password. 

Also once automatic password is done the uqiue password is automatically copied to clipboard for me to immediately paste in account creation page seamlessly and finish off the account creation process and then if I want to keep a personal record of it I can click on the Add button to update the password and account name info in a json file local to me.
